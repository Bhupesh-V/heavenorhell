# heavenorhell

It's time to book a ticket for your soul!

## Installation

1. Export necessary environment variables:
    ```bash
   export GIST_ID=""
   export GITHUB_TOKEN=""
   export TELEGRAM_CHAT_ID=""
   export TELEGRAM_TOKEN=""
    ```
2. Compile & Run
   ```bash
   go run main.go
   ```
